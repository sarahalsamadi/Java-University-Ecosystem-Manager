# 🏛️ University Ecosystem Manager

A comprehensive management system designed to simulate a full university campus environment. This project focuses on optimizing administrative workflows between departments, students, and facilities using advanced Object-Oriented Programming (OOP) structures.

## 🌟 Features

- **Comprehensive Academic Management**: Organize scientific departments, and allocate classrooms or auditoriums for specific events.
- **Human Resources Hierarchy**: Specialized classification for students (UG/PG) and staff members (Teaching/Non-Teaching).
- **Smart Logistics System**: Integrated management for the Library (lending/searching), Hostels (check-in/out), and Parking lots.
- **Resource Tracking**: Real-time monitoring of seat availability in university buses, classrooms, and food services in the canteen.
- **Robust Software Architecture**: Implementing core OOP principles like Inheritance, Encapsulation, and Composition for a scalable system.

## 🛠 Tech Stack

- **Java**: The primary language used for core logic and object-oriented structuring.
- **OOP Architecture**: Heavy reliance on Inheritance and Composition to model real-world campus relationships.
- **Collections Framework**: Efficiently managing lists of students, staff, and event schedules.
